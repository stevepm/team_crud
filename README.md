README

Team CRUD

Setup Instructions
==================
1. Clone this repository
1. `bundle`
1. Create databases: `rake db:create`
1. Run migrations: `rake db:migrate`
1. Run Specs `rake spec`
1. Run Local Server `rails s`

Admins
======
* run `rake instagram:import` to import pictures from instagram
